## FP2 讲义：二阶常微分方程

二阶常微分方程（ODE）是涉及未知函数二阶导数的方程。它们是物理系统数学建模的基础，在物理、工程和其他科学领域有无数应用。在本讲中，我们将探索求解这些方程的理论和技巧，重点关注常系数线性方程。

:::note[引出问题：弹簧-质量系统]
考虑质量为 $m$ 的物体连接在弹性系数为 $k$ 的弹簧上。如果将物体从平衡位置释放，其位置 $x(t)$ 满足：

$$
m\frac{d^2x}{dt^2} = -kx
$$

或等价地：

$$
\frac{d^2x}{dt^2} + \frac{k}{m}x = 0
$$

物体如何随时间运动？

这是二阶 ODE 的经典例子。其解描述简谐运动：$x(t) = A\cos(\omega t) + B\sin(\omega t)$，其中 $\omega = \sqrt{\frac{k}{m}}$，$A$、$B$ 是由初始条件确定的常数。

这个方程出现在弹簧以外的多种场景中，包括摆（小振幅）、电路和声波传播。
:::

## 模块 1：二次方程复习

在深入二阶 ODE 之前，我们需要复习二次方程，因为它们在求解方法中起关键作用。

**二次方程。** 形如：

$$
ax^2 + bx + c = 0
$$

的方程，其中 $a \neq 0$，$a, b, c$ 为常数。

:::note[求根公式]
$ax^2 + bx + c = 0$ 的解为：

$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

表达式 $\Delta = b^2 - 4ac$ 称为**判别式**，决定解的性质：

- $\Delta > 0$：两个不同的实根
- $\Delta = 0$：一个重实根（"双重根"）
- $\Delta < 0$：两个共轭复根
:::

## 模块 2：二阶 ODE 简介

**二阶 ODE。** 涉及未知函数二阶导数的方程。一般形式为：

$$
F\!\left(x, y, \frac{dy}{dx}, \frac{d^2y}{dx^2}\right) = 0
$$

其中 $y = f(x)$ 是未知函数。

:::note[二阶 ODE 的分类]
二阶 ODE 可以从多个角度分类：

**1. 线性 vs. 非线性**
- **线性：** 可写成 $a(x)\frac{d^2y}{dx^2} + b(x)\frac{dy}{dx} + c(x)y = g(x)$
- **非线性：** 不能写成上述形式

**2. 齐次 vs. 非齐次**
- **齐次：** $g(x) = 0$
- **非齐次：** $g(x) \neq 0$

**3. 常系数 vs. 变系数**
- **常系数：** $a(x)$、$b(x)$、$c(x)$ 都是常数
- **变系数：** 至少有一个依赖于 $x$

本讲重点是常系数线性二阶 ODE：

$$
a\frac{d^2y}{dx^2} + b\frac{dy}{dx} + cy = g(x)
$$

其中 $a$、$b$、$c$ 为常数（$a \neq 0$）。
:::

### 例 1：二阶 ODE 示例

1. $\frac{d^2y}{dx^2} + 4y = 0$ — 线性、齐次、常系数
2. $\frac{d^2y}{dx^2} + 2\frac{dy}{dx} + y = \sin x$ — 线性、非齐次、常系数
3. $x\frac{d^2y}{dx^2} + \frac{dy}{dx} + xy = 0$ — 线性、齐次、变系数
4. $\frac{d^2y}{dx^2} + \left(\frac{dy}{dx}\right)^2 + y = 0$ — 非线性
5. $\frac{d^2y}{dx^2} = y^3$ — 非线性

## 模块 3：常系数齐次线性 ODE

### 辅助方程法

我们聚焦于求解常系数齐次线性二阶 ODE：

$$
a\frac{d^2y}{dx^2} + b\frac{dy}{dx} + cy = 0
$$

:::note[辅助方程方法]
求解 $a\frac{d^2y}{dx^2} + b\frac{dy}{dx} + cy = 0$：

1. 寻找形如 $y = e^{rx}$ 的解，其中 $r$ 是待定常数

2. 代入微分方程：

$$
e^{rx}(ar^2 + br + c) = 0
$$

3. 由于 $e^{rx} \neq 0$，必须有：

$$
ar^2 + br + c = 0
$$

这就是**辅助方程**（或**特征方程**）。

4. 求解这个二次方程。通解的形式取决于根的类型。
:::

### 辅助方程的三种情况

:::note[情况 1：两个不同的实根]
若 $ar^2 + br + c = 0$ 有两个不同的实根 $r_1$ 和 $r_2$（即 $b^2 - 4ac > 0$），则：

$$
y = C_1e^{r_1x} + C_2e^{r_2x}
$$

其中 $C_1$、$C_2$ 是任意常数。

**原理：** 由于 ODE 是线性的，若 $y_1$ 和 $y_2$ 都是解，则 $C_1y_1 + C_2y_2$ 也是解。$e^{r_1x}$ 和 $e^{r_2x}$ 都满足辅助方程，这就是完整的通解。
:::

### 例 2：不同实根

求解 $\frac{d^2y}{dx^2} - 5\frac{dy}{dx} + 6y = 0$。

**解：**

1. 辅助方程：$r^2 - 5r + 6 = 0$
2. 因式分解：$(r - 2)(r - 3) = 0$
3. 根：$r_1 = 2$，$r_2 = 3$
4. 通解：$y = C_1e^{2x} + C_2e^{3x}$

:::note[情况 2：重实根]
若 $ar^2 + br + c = 0$ 有重实根 $r_1$（即 $b^2 - 4ac = 0$），则：

$$
y = C_1e^{r_1x} + C_2xe^{r_1x}
$$

**为什么出现 $xe^{r_1x}$：** 重根时需要第二个线性无关的解。可以证明 $xe^{r_1x}$ 也是解，确保通解的完整性。
:::

### 例 3：重实根

求解 $\frac{d^2y}{dx^2} - 6\frac{dy}{dx} + 9y = 0$。

**解：**

1. 辅助方程：$r^2 - 6r + 9 = 0$
2. 因式分解：$(r - 3)^2 = 0$
3. 重根：$r = 3$
4. 通解：$y = C_1e^{3x} + C_2xe^{3x}$

:::note[情况 3：共轭复根]
若 $ar^2 + br + c = 0$ 有共轭复根 $r = \alpha \pm i\beta$（即 $b^2 - 4ac < 0$），则：

$$
y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)
$$

**为什么是这种形式？** 复根产生复解：$e^{(\alpha+i\beta)x} = e^{\alpha x}(\cos\beta x + i\sin\beta x)$。由于需要实解，我们取实部和虚部的线性组合。

这种形式在物理学中特别有用，表示带指数增长或衰减的振荡运动。
:::

### 例 4：共轭复根

求解 $\frac{d^2y}{dx^2} + 4y = 0$。

**解：**

1. 辅助方程：$r^2 + 4 = 0$
2. 根：$r = \pm 2i$（即 $\alpha = 0$，$\beta = 2$）
3. 通解：$y = C_1\cos 2x + C_2\sin 2x$

:::note[练习]
求解微分方程：

$$
4\frac{d^2y}{dx^2} - 4\frac{dy}{dx} + y = 0
$$
:::

## 模块 4：非齐次二阶 ODE

掌握齐次方程后，我们转向非齐次常系数线性二阶 ODE：

$$
a\frac{d^2y}{dx^2} + b\frac{dy}{dx} + cy = g(x)
$$

其中 $g(x) \neq 0$ 是非齐次项（强迫函数）。

### 通解的结构

:::note[通解结构]
非齐次线性 ODE 的通解为：

$$
y = y_c + y_p
$$

其中：
- $y_c$ 是**余函数**（对应齐次方程 $a\frac{d^2y}{dx^2} + b\frac{dy}{dx} + cy = 0$ 的通解）
- $y_p$ 是**特积分**（非齐次方程的任意一个特解）

**要点：**
- 余函数包含任意常数
- 特积分不含任意常数
- 只需找到一个特积分
:::

### 例 5：求通解

考虑 $\frac{d^2y}{dx^2} + 4y = \sin x$。

**第 1 步：余函数 $y_c$**（来自 $\frac{d^2y}{dx^2} + 4y = 0$）：
- 辅助方程：$r^2 + 4 = 0 \implies r = \pm 2i$
- $y_c = A\cos 2x + B\sin 2x$

**第 2 步：特积分 $y_p$：**
- $y_p = -\frac{1}{3}\sin x$

**第 3 步：通解：**

$$
y = A\cos 2x + B\sin 2x - \frac{1}{3}\sin x
$$

### 待定系数法

:::note[待定系数法]
当 $g(x)$ 具有简单形式（多项式、指数、正弦、余弦或其乘积）时使用：

1. 根据 $g(x)$ 的形式，猜测 $y_p$ 的适当形式并设待定系数
2. 计算 $y_p'$ 和 $y_p''$
3. 代入原方程
4. 通过比较同类项系数求解未知系数

**重要：** 如果试探函数的任何部分已经是齐次方程的解，必须乘以 $x$（必要时乘以 $x^2$）以确保线性无关。
:::

:::note[特积分的试探函数表]

| 非齐次项 $g(x)$ | $y_p$ 的试探函数 |
|---|---|
| $k$（常数） | $A$ |
| 多项式：$a_nx^n + \cdots + a_0$ | $A_nx^n + \cdots + A_0$ |
| 指数：$ke^{\alpha x}$ | $Ae^{\alpha x}$ |
| 正弦/余弦：$k\sin\beta x$ 或 $k\cos\beta x$ | $A\sin\beta x + B\cos\beta x$ |
| $p(x)e^{\alpha x}$ | $q(x)e^{\alpha x}$（与 $p$ 同次） |
| $p(x)\sin\beta x$ 或 $p(x)\cos\beta x$ | $q_1(x)\sin\beta x + q_2(x)\cos\beta x$（同次） |
| $e^{\alpha x}\sin\beta x$ 或 $e^{\alpha x}\cos\beta x$ | $e^{\alpha x}(A\sin\beta x + B\cos\beta x)$ |

**特殊情况：** 若试探函数出现在余函数中：
- 若 $g(x) = ke^{\alpha x}$ 且 $e^{\alpha x}$ 在 $y_c$ 中，用 $y_p = Axe^{\alpha x}$
- 若 $g(x) = k\sin\beta x$ 或 $k\cos\beta x$ 且在 $y_c$ 中，用 $y_p = x(A\sin\beta x + B\cos\beta x)$
- 若重根且试探含 $xe^{\alpha x}$，用 $y_p = Ax^2e^{\alpha x}$
:::

### 例 6：多项式非齐次项

求解 $\frac{d^2y}{dx^2} - 3\frac{dy}{dx} + 2y = 4x + 5$。

**第 1 步：余函数**
- 辅助方程：$r^2 - 3r + 2 = 0 \implies r = 1, 2$
- $y_c = C_1e^x + C_2e^{2x}$

**第 2 步：特积分**
- 试探 $y_p = Ax + B$，则 $y_p' = A$，$y_p'' = 0$
- 代入：$-3A + 2(Ax + B) = 4x + 5$
- 比较系数：$2A = 4 \implies A = 2$；$2B - 6 = 5 \implies B = 5.5$
- $y_p = 2x + 5.5$

**第 3 步：通解：**

$$
y = C_1e^x + C_2e^{2x} + 2x + 5.5
$$

### 例 7：指数非齐次项

求解 $\frac{d^2y}{dx^2} - \frac{dy}{dx} - 6y = 3e^{2x}$。

**第 1 步：** 辅助方程：$r^2 - r - 6 = 0 \implies r = 3, -2$。$y_c = C_1e^{3x} + C_2e^{-2x}$。

**第 2 步：** 试探 $y_p = Ae^{2x}$。代入得 $-4A = 3 \implies A = -\frac{3}{4}$。

**第 3 步：** $y = C_1e^{3x} + C_2e^{-2x} - \frac{3}{4}e^{2x}$

### 例 8：三角函数非齐次项（特殊情况）

求解 $\frac{d^2y}{dx^2} + 4y = 3\sin 2x$。

**第 1 步：** 辅助方程：$r^2 + 4 = 0 \implies r = \pm 2i$。$y_c = C_1\cos 2x + C_2\sin 2x$。

**第 2 步：** $\sin 2x$ 和 $\cos 2x$ 已在 $y_c$ 中，乘以 $x$：

试探 $y_p = x(A\cos 2x + B\sin 2x)$。

代入比较系数：$-4A = 3 \implies A = -\frac{3}{4}$；$4B = 0 \implies B = 0$。

$y_p = -\frac{3}{4}x\cos 2x$。

**第 3 步：**

$$
y = C_1\cos 2x + C_2\sin 2x - \frac{3}{4}x\cos 2x
$$

:::note[练习]
求解微分方程：

$$
\frac{d^2y}{dx^2} - 2\frac{dy}{dx} - 3y = 4e^x + 2x
$$
:::

## 模块 5：边界条件与特解

### 初值问题

:::note[初值问题（IVP）]
二阶 ODE 的初值问题指定：
- 微分方程本身
- 函数在某点的值：$y(x_0) = y_0$
- 一阶导数在同一 $x_0$ 点的值：$y'(x_0) = y_1$

二阶 ODE 需要恰好**两个**条件来确定通解中的任意常数。
:::

### 例 9：初值问题

求解：

$$
\frac{d^2y}{dx^2} + y = x, \quad y(0) = 2, \quad y'(0) = -1
$$

**解：**

1. **余函数：** $r^2 + 1 = 0 \implies r = \pm i$，$y_c = A\cos x + B\sin x$

2. **特积分：** 试探 $y_p = Cx$。代入得 $Cx = x \implies C = 1$。$y_p = x$。

3. **通解：** $y = A\cos x + B\sin x + x$

4. **代入初值条件：**

$$
y(0) = A = 2, \quad y'(0) = B + 1 = -1 \implies B = -2
$$

5. **特解：** $y = 2\cos x - 2\sin x + x$

### 边值问题

:::note[边值问题（BVP）]
边值问题在**不同**点指定条件：

$$
y(a) = \alpha \quad \text{且} \quad y(b) = \beta
$$
:::

### 例 10：边值问题

求解：

$$
\frac{d^2y}{dx^2} + y = 0, \quad y(0) = 0, \quad y(\pi/2) = 1
$$

**解：**

1. 通解：$y = A\cos x + B\sin x$
2. $y(0) = A = 0$
3. $y(\pi/2) = B = 1$
4. $y = \sin x$

## 模块 6：二阶 ODE 的换元法

有时二阶 ODE 可以通过适当的换元来简化。我们探索两种主要类型：替换自变量 $x$ 和替换因变量 $y$。

### 变量替换（改变自变量）

:::note[替换自变量]
当微分方程包含具有共同参数的函数时，用新变量替换该参数以简化。

**链式法则变换：** 当 $t = f(x)$ 且 $y = y(t)$ 时：

$$
\frac{dy}{dx} = \frac{dy}{dt} \cdot f'(x)
$$

$$
\frac{d^2y}{dx^2} = \frac{d^2y}{dt^2} \cdot [f'(x)]^2 + \frac{dy}{dt} \cdot f''(x)
$$
:::

### 例 11：对数换元

求解：

$$
x^2\frac{d^2y}{dx^2} - 3x\frac{dy}{dx} + 4y = 0, \quad x > 0
$$

使用 $t = \ln x$。

**解：**

1. 令 $t = \ln x$，即 $x = e^t$。

2. 变换导数：

$$
\frac{dy}{dx} = \frac{dy}{dt} \cdot \frac{1}{x}
$$

$$
\frac{d^2y}{dx^2} = \frac{1}{x^2}\left(\frac{d^2y}{dt^2} - \frac{dy}{dt}\right)
$$

3. 代入原方程：

$$
\frac{d^2y}{dt^2} - 4\frac{dy}{dt} + 4y = 0
$$

4. 常系数求解：
   - 辅助方程：$(r-2)^2 = 0 \implies r = 2$（重根）
   - $y(t) = C_1e^{2t} + C_2te^{2t}$

5. 换回：

$$
y(x) = C_1x^2 + C_2x^2\ln x
$$

### 例 12：平方根换元

求解：

$$
x\frac{d^2y}{dx^2} - (6x^2 + 1)\frac{dy}{dx} + 9x^3y = x^5, \quad x > 0
$$

使用 $x = t^{\frac{1}{2}}$（即 $t = x^2$），变换为：

$$
4\frac{d^2y}{dt^2} - 12\frac{dy}{dt} + 9y = t
$$

这是常系数二阶 ODE。

### 因变量替换（变换方程）

:::note[替换因变量]
当微分方程具有特定结构时，替换因变量 $y$ 有时可以将其变换为更简单的方程。
:::

### 例 13：幂次替换

求解：

$$
\frac{d^2y}{dx^2} - \frac{(y')^2}{y} + y' = 0
$$

使用 $y = e^z$。

**解：**

1. 令 $y = e^z$，则 $y' = e^z z'$，$y'' = e^z(z')^2 + e^z z''$。

2. 代入：

$$
e^z z'' + e^z z' = 0
$$

3. 由于 $e^z \neq 0$：$z'' + z' = 0$

4. 辅助方程：$r^2 + r = 0 \implies r = 0, -1$。$z = C_1 + C_2e^{-x}$。

5. 换回：

$$
y = A \cdot e^{Be^{-x}}
$$

其中 $A = e^{C_1} > 0$，$B = C_2$ 为任意常数。

### 复习练习：对数换元

给定：

$$
x^2\frac{d^2y}{dx^2} - 2y = 1 + 4\ln x - 2(\ln x)^2, \quad x > 0
$$

证明使用 $t = \ln x$ 后变为：

$$
\frac{d^2y}{dt^2} - \frac{dy}{dt} - 2y = 1 + 4t - 2t^2
$$

然后求解以确定原方程的通解。

### 区分变换类型

:::note[因变量 vs. 自变量变换]

**因变量变换（$y \to v$）：**
- 换元重新定义因变量：$v = f(y)$，$y = g(v)$
- 导数仍对 $x$ 求导（如 $\frac{dv}{dx}$）
- 你改变的是*什么*随 $x$ 变化
- 示例：$v = y^{-2}$，$z = y^2$，$y = ve^x$

**自变量变换（$x \to t$）：**
- 换元重新定义自变量：$t = h(x)$ 或 $x = k(t)$
- $y$ 仍为因变量，但现在视为 $t$ 的函数
- 导数对 $t$ 求导（如 $\frac{dy}{dt}$）
- 你改变的是 $y$ 相对于*哪个变量*变化
- 示例：$t = \ln x$，$x = e^t$，$x = t^2$
:::

:::note[自变量变换的链式法则]

**情况 1：$t = h(x)$**

$$
\frac{dy}{dx} = \frac{dy}{dt} \cdot h'(x)
$$

$$
\frac{d^2y}{dx^2} = \frac{d^2y}{dt^2} \cdot [h'(x)]^2 + \frac{dy}{dt} \cdot h''(x)
$$

用 $x = h^{-1}(t)$ 将所有量表示为 $t$ 的函数。

**情况 2：$x = k(t)$**

$$
\frac{dy}{dx} = \frac{dy/dt}{k'(t)}
$$

$$
\frac{d^2y}{dx^2} = \frac{\frac{d^2y}{dt^2}k'(t) - \frac{dy}{dt}k''(t)}{[k'(t)]^3}
$$
:::

## 作业题

### 第 1 题

(a) 求下列方程的通解：

$$
\frac{d^2y}{dx^2} + 2\frac{dy}{dx} + 5y = 6\cos x
$$

(b) 求满足 $x = 0$ 时 $y = 0$ 且 $\frac{dy}{dx} = 0$ 的特解。

### 第 2 题

(a) 求下列方程的通解：

$$
\frac{d^2y}{dx^2} - 6\frac{dy}{dx} + 8y = 2x^2 + x
$$

(b) 求满足 $x = 0$ 时 $y = 1$ 且 $\frac{dy}{dx} = 0$ 的特解。

### 第 3 题

(a) 证明变换 $y = xv$ 将方程

$$
3\frac{d^2y}{dx^2} - \frac{6}{x}\frac{dy}{dx} + \frac{6y}{x^2} + 3y = x^2, \quad x \neq 0 \quad \text{(I)}
$$

变换为

$$
3\frac{d^2v}{dx^2} + 3v = x \quad \text{(II)}
$$

(b) 由此求 (I) 的通解，答案写成 $y = f(x)$ 的形式。

### 第 4 题

(a) 证明变换 $x = t^2$ 将方程

$$
4x\frac{d^2y}{dx^2} + 2(1 + 2\sqrt{x})\frac{dy}{dx} - 15y = 15x \quad \text{(I)}
$$

变换为

$$
\frac{d^2y}{dt^2} + 2\frac{dy}{dt} - 15y = 15t^2 \quad \text{(II)}
$$

(b) 求解 (II)，用 $t$ 表示 $y$。

(c) 由此求 (I) 的通解。
